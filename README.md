# 🛍️ Motisa - E-commerce Frontend

A modern and responsive frontend-only e-commerce website built with React and Vite. The project fetches products from the Fake Store API and demonstrates routing, state management, authentication UI, cart functionality, wishlist, pagination, and responsive design.

## 🚀 Live Demo

https://motisa-e-commerce.vercel.app/

## 📂 GitHub Repository

https://github.com/abpandey4/Motisa-E-commerce

## ✨ Features

- Responsive UI
- Product Listing
- Product Details
- Search Products
- Login Page
- Cart
- Wishlist
- Pagination
- 404 Not Found Page
- React Router Navigation
- Fake Store API Integration
- Loading & Error Handling
  
## 🛠 Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS
- React Router DOM
- Fake Store API
